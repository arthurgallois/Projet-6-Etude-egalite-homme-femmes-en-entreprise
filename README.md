# Projet-6-Etude-egalite-homme-femmes-en-entreprise
Projet d'étude de l'état de l'égalité Hommes-Femmes en entreprise via Knime
Les problématiques sont les suivantes : 
- A partir des données initiales, effectuer le calcul des différents indicateurs d'égalité hommes-femmes afin de déterminer la nécessité d'actions correctives
- Nettoyer les données pour qu'elles soient conformes à la legislation RGPD

L'outil utilisé est Knime, du filtrage des données au calcul des indicateurs
L'étude a révélé des inégalités détaillées dans le Powerpoint joint.
